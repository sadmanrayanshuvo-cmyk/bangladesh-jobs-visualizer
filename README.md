# Bangladesh Jobs Visualizer

Interactive Bangladesh job market treemap using BBS LFS 2024, BSCO 2020, and a separate modeled AI exposure layer.

## Purpose
This project visualizes Bangladesh occupations by:
- employment size
- wage tier
- education tier
- modeled AI exposure

## Project structure
- `docs/` — published GitHub Pages files
- `data/` — CSV and JSON data files
- `src/` — source frontend files

## Data notes
- Employment structure should be anchored to BBS Labour Force Survey 2024
- Occupation labels and codes follow BSCO 2020
- AI exposure is a modeled estimate, not an official BBS statistic
